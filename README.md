# TareaApp 📚

App de gestión de tareas escolares para I.E. Del Pinar.

## Descargar

Descarga la última versión del APK en [Releases](https://github.com/kraqinc/TareaApp/releases).

## Funciones

- Iniciar sesión como estudiante o profesor
- Ver tareas por grado, materia y período
- Profesores publican tareas con archivos adjuntos
- Estudiantes entregan tareas desde la app
- Notificaciones push al entregar y revisar
- Un profesor por materia por grado
- Primaria (1°–5°) y Bachillerato (6°–9°)
